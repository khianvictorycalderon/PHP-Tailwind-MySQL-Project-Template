# PHP + Tailwind CSS + MySQL Project Template
A boilerplate for a full-stack PHP + MySQL Project

## Prerequisites
- Apache
- MySQL Server
I recommend using XAMPP as it comes with built-in Apache for PHP and MySQL Server.

## Setup
1. Go to you XAMPP htdocs directory (usually in `C:\xampp\htdocs`, depending on where you installed your XAMPP)
2. Clone this repository by running `git clone https://github.com/khianvictorycalderon/PHP-Tailwind-MySQL-Project-Template`
3. Start Apache and MySQL from XAMPP control panel.
